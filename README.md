### Cloning the repository

```shell
git clone https://github.com/bintangnugrahaa/ai-image-generation.git
```

### Install packages

```shell
npm install
```

### Setup .env file


```js
VITE_RAPID_API_ARTICLE_KEY=
```

### Start the app

```shell
npm run dev
```
